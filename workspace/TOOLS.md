# Available Tools

## File Operations

- **read** — Read file contents
- **write** — Create or overwrite files
- **edit** — Make precise find-and-replace edits
- **multi_edit** — Apply multiple edits to a single file
- **grep** — Search file contents for patterns
- **find** — Find files by glob pattern
- **ls** — List directory contents

## Shell Execution

- **exec** — Run shell commands (sandboxed when enabled)

## Web Access

- **web_search** — Search the web (Brave API)
- **web_fetch** — Fetch and extract readable content from a URL

## Browser

- **browser** — Interactive Puppeteer browser (navigate, click, type, screenshot)

## File Operations (Extended)

- **file_ops** — Download URLs, move/copy/delete files, create directories

## Memory

- **memory** — Persistent memory across conversations (store/search/list/update/delete)

## Scheduling

- **cron** — Schedule reminders and recurring jobs
